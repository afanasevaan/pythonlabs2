# pyhtonlabs2
## Лабораторная работа №1: Класс и инкапсуляция

**Цели:** Освоить создание классов, инкапсуляцию, свойства (@property) и магические методы в Python.

**В ходе выполнения лабораторной работы были успешно закреплены на практике следующие навыки:**

- Создание класса с закрытыми атрибутами
- Реализация конструктора с валидацией данных
- Использование @property для геттеров и сеттеров
- Переопределение магических методов str, repr, eq
- Работа с атрибутами класса и экземпляра
- Реализация бизнес-методов и логики состояния объекта
- Обработка исключений при некорректных данных

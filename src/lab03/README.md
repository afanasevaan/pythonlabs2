## 2. Описание реализованной иерархии классов

### Иерархия

```text
Book
├── PrintedBook (печатная книга)
├── Ebook (электронная книга)
└── AudioBook (аудиокнига)
```
## Описание иерархии классов

**Базовый класс (Book)** — из ЛР-1: title, author, year, pages, price, is_available. Методы: borrow(), return_book(), get_age_category().

**Дочерние классы:**

| Класс | Новые атрибуты | Новый метод |
|-------|----------------|-------------|
| PrintedBook | binding (переплёт), circulation (тираж) | get_shipping_weight() |
| Ebook | file_format (формат), file_size_mb (размер) | get_download_time() |
| AudioBook | duration_hours (длит-ть), narrator (чтец) | get_remaining_time() |

**Различия:** печатная — физическая книга с весом; электронная — файл со временем скачивания; аудио — запись с длительностью прослушивания.

## Сценарий 1: Создание объектов разных типов

Созданы три объекта:
- **PrintedBook** — "Война и мир" (твердый переплёт, тираж 5000)
- **Ebook** — "1984" (формат EPUB, 2.5 МБ)
- **AudioBook** — "Мастер и Маргарита" (читает Хабенский, 12.5 ч)

Каждый объект выводится в своём формате благодаря переопределённому методу `__str__`.

## Сценарий 3: Уникальные методы

Вызваны методы, специфичные для каждого дочернего класса:

- **PrintedBook** — `get_shipping_weight()`: вес книги "Война и мир" составил 13.00 кг (1300 страниц × 0.01 кг)

- **Ebook** — `get_download_time()`: время скачивания "1984" — 2.00 секунды (при скорости 10 Мбит/с)

- **AudioBook** — `get_remaining_time()`: осталось слушать "Мастер и Маргарита" — 9.0 часов (из 12.5 прослушано 3.5)

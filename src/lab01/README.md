# Лабораторная работа №1: Класс и инкапсуляция

## Вариант: Библиотека / Книги

## Описание проекта

Реализация класса `Book` (Книга) для библиотечной системы. 
Класс демонстрирует принципы ООП: инкапсуляцию, свойства, магические методы и бизнес-логику.

## Структура класса

### Атрибуты (закрытые поля)
- `_title` - название книги (строка)
- `_author` - автор книги (строка)  
- `_year` - год издания (целое число)
- `_pages` - количество страниц (целое число)
- `_price` - стоимость книги (число с плавающей точкой)

### Атрибуты класса
- `MAX_PRICE = 10000` - максимальная допустимая цена
- `MIN_YEAR = 1400` - минимальный год издания

### Состояние объекта
- `is_available` - доступность книги (True/False)

### Методы

#### Магические методы
- `__init__` - конструктор с валидацией данных
- `__str__` - строковое представление для пользователя
- `__repr__` - техническое представление для разработчика
- `__eq__` - сравнение книг по названию и автору

#### Свойства (property)
- `title` - геттер и сеттер с валидацией
- `author` - геттер и сеттер с валидацией
- `year` - геттер и сеттер с валидацией
- `pages` - геттер и сеттер с валидацией
- `price` - геттер и сеттер с валидацией

#### Методы валидации
- `_validate_title` - проверка названия
- `_validate_author` - проверка автора
- `_validate_year` - проверка года
- `_validate_pages` - проверка страниц
- `_validate_price` - проверка цены

#### Бизнес-методы
- `get_age_category()` - определение возрастной категории книги
- `apply_discount(percent)` - применение скидки
- `is_expensive()` - проверка на дорогую книгу
- `borrow()` - выдача книги (изменение состояния)
- `return_book()` - возврат книги (изменение состояния)
